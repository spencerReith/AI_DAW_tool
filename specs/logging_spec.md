# Logging Spec

## Background
Implement logging for the application. This should be continued appendeges with timestampt to a TXT file.

## Goals

### On the StableAudio Side

Log the user prompt, log the bpm, duration, and versions requested
Log Claudes reworded prompt(s). log the name(es) of the outputted audio.

### On the MusicGen Side

Log the user prompt, input file name.
Log the name of the outputted "full" mp3 file that is synthesized from combination of melody and chord midi, and the the name of the file outputted by musicGen
